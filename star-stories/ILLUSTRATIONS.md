# Illustration strategy

## Now (v1, in the books): house-style vector art

Hand-drawn **SVG line art** embedded in the HTML — gold (#c9a227) and ink
(#2c2416) on warm white, with one accent color per book (Lars: sea-teal,
Luis: grass-green, Nova: moon-silver). Vector = perfectly sharp at any print
size, tiny files, and a consistent style across the whole series — it *is*
the brand. Every spread has a vignette; covers carry a full scene.

## Later options (richer editions)

1. **Human illustrator** — the premium product route. `ARTIST-BRIEF.md` in this
   folder can be handed to any illustrator (Fiverr, local art student, pro):
   it fixes palette, mood, and the per-page shot list so the style stays ours.
2. **AI-generated art** directed by you (Midjourney / DALL·E / Ideogram) — use
   the same brief as the prompt base; keep one seed/style reference per book so
   pages match. I integrate the images and re-typeset.
3. **Hybrid** — keep the SVG house style for the series, commission full art
   only for the cover.

Recommendation: ship v1 with the SVG house style (done), test with the real
readers (the children), then decide if a book earns the illustrator.
